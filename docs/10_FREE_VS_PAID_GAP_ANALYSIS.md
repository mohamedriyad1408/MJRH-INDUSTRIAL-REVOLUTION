### 🟢 1. دعم عملات متعددة (Multi-Currency)

| البند | التفاصيل |
|---|---|
| **الوصف** | إضافة USD و EUR و SAR و AED كعملات مدعومة بجانب EGP |
| **التكلفة** | $0 — تغييرات في الكود فقط |
| **الملفات** | `lib/format.ts`, `styles.css`, `app_settings`, `orders` table |
| **الجهد** | 2-3 أسابيع تطوير |
| **التفاصيل** | إضافة حقل `currency` في `app_settings`، تعديل `fmtMoney()` لدعم عملات متعددة، إضافة التنسيق الصحيح لكل عملة ($1,234.56 / €1.234,56 / ١٬٢٣٤٫٥٦ ر.س) |

**التنفيذ المطلوب:**
```ts
// lib/format.ts — إضافة دعم العملات
const CURRENCY_CONFIG = {
  EGP: { symbol: 'ج.م', locale: 'ar-EG', position: 'after' },
  USD: { symbol: '$', locale: 'en-US', position: 'before' },
  EUR: { symbol: '€', locale: 'de-DE', position: 'before' },
  SAR: { symbol: 'ر.س', locale: 'ar-SA', position: 'after' },
  AED: { symbol: 'د.إ', locale: 'ar-AE', position: 'after' },
};
```

### 🟢 3. Real-Time WebSocket Updates

| البند | التفاصيل |
|---|---|
| **الوصف** | تحديث فوري لشاشات المحطات بدل الـ polling كل 60 ثانية |
| **التكلفة** | $0 — Supabase Realtime مجاني على الخطة المجانية |
| **الملفات** | `components/station-board.tsx`, `routes/$tenant/stations/*.tsx` |
| **الجهد** | 2-3 أسابيع |
| **التفاصيل** | استخدام `supabase.channel()` للاستماع لتغييرات `orders` و `service_units` |

**التنفيذ:**
```ts
// مثال: تحديث فوري لشاشة المحطة
supabase
  .channel('station-updates')
  .on('postgres_changes', {
    event: '*',
    schema: 'public',
    table: 'orders',
    filter: `tenant_id=eq.${tenantId}`,
  }, (payload) => {
    // تحديث الـ UI فوراً
    load();
  })
  .subscribe();
```

| البند | التفاصيل |
