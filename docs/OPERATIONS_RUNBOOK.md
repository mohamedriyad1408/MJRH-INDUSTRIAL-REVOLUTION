# MJRH Operations Runbook

## Daily start

1. Open **تشغيل اليوم**.
2. Confirm **جاهزية العمل الآن**.
3. Run **فحص النظام** if there are warnings.
4. Ensure drivers update their location.
5. Review WhatsApp outbox.

## During the day

- Reception creates and routes orders.
- Cleaning handles cleaning and customer/internal returns.
- Drying/assembly resolves label issues.
- Ironing is assigned only to present technicians.
- Packing and QC must pass before delivery.
- Payment proofs must appear as order attachments.

## End of day

1. Review financial audit.
2. Close cash safes.
3. Save daily report.
4. Resolve or document any remaining APDO issue.

## If a worker sees old tasks

1. Check if the order is delivered/cancelled.
2. Open System Health and client error logs.
3. Use APDO repair if needed.
4. If task remains, report order number and label code.

## Customer return after delivery

Do not reopen the entire delivered order.

Use:

```txt
Order page → Piece → مرتجع عميل
```

Then follow the return through the relevant station.
