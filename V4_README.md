# MJRH V4 Sovereign Platform

## ⚠️ Important Note
Layer V4 is a **Greenfield Platform**. It is strictly reserved for **NEW Tenants** created via the Onboarding Wizard.

### Migration Policy:
- **Legacy Tenants** (e.g., `dry-tech`, `laundry-showcase`) will continue to operate on the stable **V2/V3 Core** (Schema: `public`).
- **NO Migration** of legacy data to V4 schemas (`v4_l1`, `v4_l2`, etc.) is permitted to ensure zero downtime and maximum stability for existing businesses.
- V4 schemas will remain pure and empty until a new sovereign enterprise is provisioned.

### Architecture:
- V4 uses a **Multi-Layered Sovereign Model** (L1-L10).
- V2/V3 uses the **Classic RLS Model**.
