# دليل مدير المنصة (Super Admin Guide)

بصفتك مدير المنصة، لديك الصلاحية الكاملة للتحكم في كافة المشاريع والمستأجرين.

## 1. إدارة المشاريع (Tenants)
- **تأسيس مشروع جديد:** من صفحة "إدارة المشاريع"، يمكنك إضافة مشروع جديد، تحديد هويته البصرية، وإنشاء حساب المالك الأول له.
- **تفعيل/إيقاف:** يمكنك تجميد أي مشروع سحابي فورياً في حال انتهاء الاشتراك أو مخالفة الشروط.
- **صيانة النواة:** زر "صيانة النواة" يقوم بإعادة فحص وتجهيز الإعدادات الافتراضية (الخزنة، شجرة الحسابات) لأي مشروع.

## 2. إدارة المستخدمين
- يمكنك عرض جميع المستخدمين المسجلين في المنظومة عبر كافة المشاريع.
- تغيير كلمات المرور لأي مستخدم في حال نسيانها.
- حذف الحسابات غير النشطة.

## 3. مراقبة الصحة (System Health)
- تتبع الـ Uptime وقواعد البيانات.
- مراقبة رسوم المنصة والاشتراكات المستحقة.

---
*تنبيه أمني: حافظ على سرية حساب السوبر أدمن، حيث يمنح وصولاً كاملاً لكافة بيانات العملاء والماليات.*
