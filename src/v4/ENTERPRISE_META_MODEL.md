# MJRH V4 — Universal Enterprise Operating Model (UEOM)

## 1. Sovereignty Layer (Core)
- **Unit:** Org-Node (Recursive Tree).
- **Responsibility:** Identity, Legal Boundary, Authority Delegation.
- **Constraints:** No business logic, only structure.

## 2. Governance Layer (Platform)
- **Unit:** Policy, Goal, Risk, Compliance.
- **Responsibility:** Guardrails for execution, Objective setting.
- **Mechanism:** Intercepts Execution Layer events.

## 3. Capability Layer (Capability Packs)
- **Unit:** Competency, Capacity, Resource.
- **Responsibility:** Provisioning of Human, Machine, Financial, and Knowledge assets.

## 4. Execution Layer (Core)
- **Unit:** Value Stream, Activity, Event.
- **Responsibility:** Turning resources into output through state-governed processes.
- **Source of Truth:** Immutable Event Stream.

## 5. Integrity Layer (Platform)
- **Unit:** Artifact, Ledger, Indicator (KPI).
- **Responsibility:** Corporate memory, Auditability, and Performance measurement.
